# Brand and Trademark Policy

DOCSight is open source software. The code is licensed separately under the MIT License.

This policy covers the use of the `DOCSight` name, logo, and project branding. Code rights and brand rights are not the same thing.

## Why This Exists

DOCSight is intended to help users document internet issues and put pressure on ISPs with solid evidence. The project accepts voluntary support, but the code remains open.

To keep that mission clear, community use of the code is welcome, but uses of the DOCSight name and branding must not imply official approval where none exists.

## Allowed Without Asking

You may:

- refer to DOCSight by name when discussing, reviewing, packaging, or linking to the project
- say that your fork, module, image, or device is "based on DOCSight" or "compatible with DOCSight"
- use the unmodified project name to point to the upstream repository, documentation, or releases
- write tutorials, reviews, benchmarks, and comparison content about DOCSight

## Not Allowed Without Permission

You may not:

- market a fork, appliance, hosted service, or commercial offer as the official DOCSight product
- use the DOCSight logo or branding in a way that suggests endorsement, certification, partnership, or approval by the upstream project
- sell hardware, preinstalled systems, or paid services under the DOCSight name without prior permission
- use confusingly similar branding that is likely to make users think your offer is the upstream project

## Forks and Redistributors

Forks and redistributors are welcome. If you distribute modified versions, please make the distinction clear.

Recommended wording:

- "Fork of DOCSight"
- "Based on DOCSight"
- "Compatible with DOCSight"

Avoid:

- "Official DOCSight Appliance"
- "DOCSight Certified"
- "DOCSight Support Team"

unless you have explicit written permission.

## Commercial and OEM Use

If you want to bundle DOCSight with hardware, offer a paid managed service around it, or discuss an official partnership, please contact the maintainer first.

Commercial use of the code may be allowed by the software license, but use of the DOCSight name, logo, and official branding still requires permission.

## Reporting Misleading Use

If you see a product, service, or website that appears to misuse the DOCSight name or logo, open a GitHub discussion or contact the maintainer through the project profile.

## Scope

This policy is intended to protect users from confusion and protect the project's identity. It does not modify the underlying software license.
